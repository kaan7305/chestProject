Interactive C++ Chess Game

About This Project

This is a fully-featured command-line chess game implemented in C++ with:

  • Complete chess rule implementation
  • Check, checkmate, and stalemate detection
  • Game save/load functionality
  • Material point value tracking

4. Checkmate Position

Checkmate - Game Over!
Black's king is in check from White's queen on F7, and has no legal moves to escape:
  • King cannot move to any adjacent square (all controlled by White's pieces)
  • No piece can block the attack
  • The attacking queen cannot be captured
This is a classic Scholar's Mate pattern.

How to Use

Compilation

make

Running the Game

./chess

Commands

  • M <move> - Make a move (e.g., M E2E4)
  • L <filename> - Load a saved game
  • S <filename> - Save the current game
  • ? - Show help menu
  • Q - Quit the game

Move Format

Moves are specified as 4 characters:

  • First 2 characters: Starting position (column A-H, row 1-8)
  • Last 2 characters: Ending position (column A-H, row 1-8)

Example: M E2E4 moves the piece from E2 to E4
